# Project 1: Rule-Based AI Chatbot

## Overview
This project is a simple rule-based AI chatbot created using Python.  
The chatbot responds to predefined user inputs using dictionary-based matching and basic control flow.

## Objective
The goal of this project is to understand the foundation of Artificial Intelligence through rule-based decision-making logic.

## Features
- Handles greetings
- Answers basic AI-related questions
- Uses a dictionary as a knowledge base
- Sanitizes user input using lower() and strip()
- Provides fallback response for unknown inputs
- Runs continuously using a while loop
- Supports clean exit commands

## Technologies Used
- Python
- Control Flow
- Dictionaries
- Functions
- While Loop
